1. Introdução

O objetivo deste documento é analisar quais perguntas do NPS (Net Promoter Score) mais influenciam a classificação dos clientes como detratores, neutros ou promotores.

Esta análise ajudará a identificar os fatores determinantes para melhorar a experiência do cliente e aprimorar estratégias de mercado.

2. Desbalanceamento da Base de Dados

Para a análise, a base de dados foi desbalanceada em diferentes níveis:

Base Total: Dados gerais sem segmentação.

Base Brasil: Dados filtrados por território nacional.

Base Grupo 4: Dados filtrados por características específicas do grupo.

Base Grupo 4 por Regiões: Dados do Grupo 4 segmentados por localização geográfica.

Base Grupo 4 por Safra: Dados do Grupo 4 organizados por período de safra.

3. Correlação de Spearman

A correlação de Spearman foi utilizada para identificar quais perguntas do NPS têm maior relação com a classificação dos clientes. Este método é ideal para detectar relações monótonas, independentemente de linearidade.

Os resultados são apresentados em gráficos de barras, destacando as correlações para cada pergunta e sua influência no NPS.

7. Conclusão

A análise revelou que as perguntas mais influentes no NPS foram:

disponibilidade e confiabilidade mecánica (csat) : Principal fator para os não promotores.

disponibilidade e confiabilidade mecánica (csat) : Influência significativa em detratores.

capacidade operacional (hectares por hora) (csat) : Influência significativa em neutros.

Recomenda-se reavaliar a ordem das perguntas no formulário, priorizando questões menos respondidas, como conforto, para explorar novos insights. Estratégias de melhoria podem incluir:

Foco em confiabilidade mecânica e conforto, que se destacaram como fatores diferenciais.

Adoção de novas métricas para avaliar tendências regionais e sazonais.

Essa análise ajudará a refinar a coleta de dados e melhorar a experiência do cliente..
